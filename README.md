# 2D Tank Battle Game 🚀

A robust 2D tank combat game developed in **Java**, focusing on clean code practices and Core OOP principles. This project features player-controlled mechanics vs. scripted enemy units.

---

### 🛠 Technical Highlights
- **Engine:** Built from scratch using Java (Swing/AWT) for graphics and game loop management.
- **OOP Architecture:** - **Inheritance:** Base entity classes for tanks, bullets, and power-ups.
  - **Encapsulation:** Secure state management for game entities.
  - **Polymorphism:** Unified handling of different enemy movement patterns.
- **Collision Detection:** Custom-built logic for bounding-box interactions between tanks and environment.

### 🎮 Features
- **Player Mechanics:** Responsive movement and cooldown-based firing system.
- **Automated Enemies:** Scripted AI patterns for patrol and attack behaviors.
- **Map System:** Collision-enabled boundaries and obstacles.
